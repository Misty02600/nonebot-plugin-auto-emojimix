# Product Context - nonebot-plugin-auto-emojimix

## 为什么存在
Google Emoji Kitchen 提供了将两个 emoji 合成为一张新图片的有趣功能。本插件将这一功能带入 QQ 群聊/私聊场景，让用户无需打开浏览器即可快速获取 emoji 合成图。

## 解决的问题
- 用户想要在聊天中使用配对 emoji 表情包
- 手动前往 Google Emoji Kitchen 网站不方便
- 需要一个自动、即时的 emoji 合成解决方案

## 工作方式
1. 用户在群聊/私聊中发送 `emoji1+emoji2` 格式的消息
2. 插件通过正则表达式匹配消息中的 emoji 对
3. 使用本地 JSON 数据查找组合是否存在
4. 如果存在，构建图片 URL 并异步下载
5. 将合成图片作为消息发送回群聊/私聊

## 用户体验目标
- **零学习成本**: 直接发送 emoji+emoji 即可，无需额外指令
- **快速响应**: 本地数据查找 + 异步下载，响应迅速
- **容错友好**: 不支持的组合会给出清晰提示
- **透明代理**: 支持 HTTP 代理配置，解决网络访问问题
